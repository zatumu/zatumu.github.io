# 材料

バター：薄力粉：牛乳＝1:1:10

- 薄力粉 50g
- バター 50g
- 牛乳 500ml
- 塩、白こしょう、ナツメグ…各適量
# 手順

1. バターを溶かし、薄力粉を加え、弱火でなめらな感触になるまで加熱する
2. 一度火を止めて、粗熱を取る
3. 牛乳をすこしだけ温めておく
4. ルーを弱火にかけ、泡立て器でかき混ぜながら、少しずつ牛乳を加える

# そのほか

マーガリンで、かつ、適当な小麦粉でやってもできた。分量の比率は守るべきだ。融通の効かない電気コンロでやるときは、スキレットの上にミルクパンを置いて、二重鍋でやると焦さずにできた。
